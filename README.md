## ☁️ About AWS re/Start
**AWS re/Start** is a full-time, skills development programme designed by Amazon Web Services (AWS).  
It prepares learners for cloud computing careers by combining **technical training**, **hands-on labs**, and **employability skills**.  

Key learning areas include:
- Cloud computing fundamentals
- AWS core services (EC2, S3, RDS, IAM, Lambda)
- Storage and file systems in the cloud
- Linux, networking, and security
- Databases and data management
