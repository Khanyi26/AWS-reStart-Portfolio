# 3D E-Commerce AWS Architecture

## Overview
This project demonstrates a cloud architecture for a next-generation 3D e-commerce platform. 
It includes AWS service selection, design rationale, architecture diagram, and explanation of trade-offs.

## Files
- `3d-ecommerce-architecture.docx` – Detailed explanation and architecture diagram
- `3D AWS Diagram` – Visual architecture diagram exported from Draw.io

## Key Features
- High Availability (Multi-AZ, failover, load balancing)
- Scalability (EC2 Auto Scaling, AWS Lambda, DynamoDB scaling)
- Performance (CloudFront, S3 caching)
- Security (IAM, WAF, HTTPS)
- Cost Optimization (Auto-scaling, pay-per-use services)

## AWS Services Used
- Amazon S3
- Amazon CloudFront
- Route 53
- EC2 Auto Scaling
- Application Load Balancer
- AWS Lambda
- DynamoDB
- RDS
- CloudWatch
- WAF & IAM
